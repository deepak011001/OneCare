# Publishing Workflow

States: `draft` → `review` → `approved` → `published` → `archived` | `expired`.

Approvals: request → decide (approve/reject). Settings `requireApproval` gates publish.

APIs: publish, archive, approve-request, approvals decide  
UI: Documents · Approvals · Settings  
Contract: [`KNOWLEDGE_ADMINISTRATION.md`](./KNOWLEDGE_ADMINISTRATION.md)
