# Document Management

Admin documents store title, body, summary, format, status, versions, categories, tags, ACL, fingerprints, token/chunk estimates. Formats: markdown, html, richtext, pdf, docx, pptx, xlsx, csv, txt, image (binary pipeline next).

APIs: `/v1/admin/knowledge/documents*`  
UI: `/knowledge/documents` · `/knowledge/uploads`  
Contract: [`KNOWLEDGE_ADMINISTRATION.md`](./KNOWLEDGE_ADMINISTRATION.md)
